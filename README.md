### Email:
Currently only works in development.

1. Create a .env file

        GMAIL_DOMAIN=gmail.com
        GMAIL_USERNAME=example@gmail.com
        GMAIL_PASSWORD=ExamplePassword

2. run `bundle install`
3. `foreman start` (instead of `rails s`)
4. send all da emails
